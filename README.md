# mobile-sales-analysis
Mobile Sales &amp; Revenue Analysis using SQL and Excel

## Mobile Sales & Revenue Analysis (SQL + Excel)

This project analyzes mobile phone sales data using SQL for data extraction and Excel for dashboard visualization.

### Tools Used
- SQL (MySQL)
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard Design

### SQL Work
- Calculated delivery duration using DATEDIFF
- Filtered delayed deliveries (>7 days)
- Sorted records to identify longest delivery times

### Dashboard Insights
- Top revenue-generating brand
- Region-wise revenue contribution
- Brand-wise revenue comparison
- Units sold by RAM configuration
- Revenue trends by RAM
